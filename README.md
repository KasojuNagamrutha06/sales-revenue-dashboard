\# 🍫 Sales \& Revenue Analysis Dashboard



\## 🚀 Overview

This project is a fully interactive \*\*Sales \& Revenue Analysis Dashboard\*\* built using Python and Streamlit.  

It provides actionable insights into business performance through dynamic visualizations, KPIs, and trend analysis.



The dashboard enables users to monitor revenue, analyze sales patterns, and make data-driven decisions efficiently.



\---



\## 🎯 Objectives

\- Analyze sales and revenue performance

\- Identify growth trends and seasonal patterns

\- Evaluate product and regional performance

\- Enable interactive business insights



\---



\## 🛠️ Tech Stack

\- \*\*Python\*\* – Core programming

\- \*\*Pandas\*\* – Data cleaning \& analysis

\- \*\*Plotly\*\* – Interactive visualizations

\- \*\*Streamlit\*\* – Web app deployment



\---



\## 📊 Key Features



\### 🔹 KPI Metrics

\- Total Revenue

\- Total Sales (Boxes Shipped)

\- Revenue Growth (%)

\- Sales Growth (%)

\- Average Order Value



\### 🔹 Interactive Visualizations

\- 📈 Monthly Revenue Trend

\- 📦 Sales Trend Analysis

\- 📊 Product-wise Performance

\- 🌍 Country-wise Revenue Distribution

\- 👤 Salesperson Performance



\### 🔹 Advanced Insights

\- 🏆 Top 5 Performing Products

\- 📅 Best Revenue Month Identification

\- 📉 Growth Trend Analysis



\### 🔹 User Controls

\- Dynamic filters for:

&#x20; - Country

&#x20; - Product
